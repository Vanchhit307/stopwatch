# ⏱️ Java Stopwatch (GUI)

A simple and user-friendly **Stopwatch application** built using **Java Swing (AWT + GUI)**.
This project demonstrates event handling, timers, and GUI design in Java.

---

## 🚀 Features

* ▶️ **Start** the stopwatch
* ⏸️ **Stop/Pause** the stopwatch
* 🔄 **Reset** the stopwatch to zero
* 🏁 **Lap Time Recording**
* 📜 Scrollable list of lap times
* 🎨 Clean and dark-themed user interface

---

## 🖥️ Tech Stack

* Java
* Swing (GUI)
* AWT
* Event Handling
* Timer Class

## ⚙️ How It Works

* The stopwatch uses a **Swing Timer** to update time every 10 milliseconds.
* Time is displayed in **MM:SS:MS format**.
* Lap times are stored and displayed in a text area.
* Buttons trigger actions using **event listeners**.
